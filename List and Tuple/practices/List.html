Introduction to Lists
A list is an ordered, mutable collection that stores multiple values. Lists can contain duplicate values and different data types.
  • Ordered - Elements have a specific position
  • Mutable - Can be changed after creation
  • Allow Duplicates - Same values can appear multiple times
  • Mixed Types - Can store any data type
fruits = ["Apple", "Banana", "Mango"]
print(fruits)
['Apple', 'Banana', 'Mango']

List Slicing
Extract portions of a list using slice notation: list[start:end:step]
Basic Slicing [start:end]: 
nums = [0, 1, 2, 3, 4, 5]
print(nums[1:4])  # Index 1 to 3 (4 excluded)
[1, 2, 3]
Reverse [::-1]: 
print(nums[::-1])  # Reverse entire list
[5, 4, 3, 2, 1, 0]
Step Slicing: 
print(nums[::2])  # Every 2nd element
print(nums[1:5:2])  # From 1 to 5, step 2
[0, 2, 4]
[1, 3]

Adding Elements
Add new elements to a list using append(), extend(), or insert().

fruits = ["Apple", "Banana"]

List Methods Reference
Complete table of all list methods with syntax and descriptions.
Method Syntax Description
append() list.append(item) Adds a single item to the end
extend() list.extend(iterable) Adds multiple items from an iterable
insert() list.insert(index, item) Inserts item at specified position

List Comprehension
Create lists concisely using a single line of code. More efficient than loops.
Basic Comprehension: 
squares = [x**2 for x in range(5)]
print(squares)
[0, 1, 4, 9, 16]
With Condition (if): 
evens = [x for x in range(10) if x % 2 == 0]
print(evens)
[0, 2, 4, 6, 8]
Nested Comprehension: 
matrix = [[i*j for j in range(3)] for i in range(3)]
print(matrix)
[[0, 0, 0], [0, 1, 2], [0, 2, 4]]

Copying Lists
Different methods of copying with different behaviors.
Assignment (Reference): 
list1 = [1, 2, 3]
list2 = list1  # Same reference
list2[0] = 99
print(list1)  # [99, 2, 3] - Changes affect both!
[99, 2, 3]
Shallow Copy - copy(): 
list1 = [1, 2, 3]
list2 = list1.copy()
list2[0] = 99
print(list1)  # [1, 2, 3] - Original unchanged
[1, 2, 3]
Shallow Copy - list() & [:]: 
list3 = list(list1)  # or list1[:]
# Both create shallow copies

Sorting Lists
Sort list elements using sort() or sorted() method.
sort() - In-place:

nums.sort()

sorted() - New List: 
sorted_nums = sorted(nums, reverse=True)

Sort with Key: 
words = ["apple", "pie", "zoo", "my"]
words.sort(key=len)  # Sort by length
print(words)  # ['my', 'zoo', 'pie', 'apple']
['my', 'zoo', 'pie', 'apple']

Flatten Nested List: 
nested = [[1, 2], [3, [4, 5]]]
def flatten(lst):
    result = []
    for item in lst:
        if isinstance(item, list):
            result.extend(flatten(item))
        else:
            result.append(item)
    return result
Check Palindrome List: 
def is_palindrome(lst):
    return lst == lst[::-1]
print(is_palindrome([1, 2, 3, 2, 1]))  # True
